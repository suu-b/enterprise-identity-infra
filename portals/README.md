# Portals

This directory contains the simulated applications that Wolke Systems will eventually protect, federate with, or govern through its identity infrastructure.

## Wolke Systems Internal Portals

### HR Portal

Used by HR employees and managers to manage employee profiles, onboarding, benefits, time-off approvals, contractor lifecycle reviews, and manager-facing people workflows.

### Finance Dashboard

Used by Finance teams to review budgets, payroll exceptions, reimbursements, spending reports, and approval-sensitive financial actions.

### Internal Git Service

Used by Engineering to manage repositories, pull requests, protected branches, deployment keys, and code ownership across internal software projects.

### Admin Console

Used by Security and system administrators to manage organization settings, application registration, user operations, temporary access grants, and administrative audit visibility.

### Analytics Platform

Used by managers, analysts, operations, and security reviewers to inspect cross-department metrics, reporting dashboards, access trends, and governance-oriented data.

## External Portal

### WorkYear

WorkYear is a Workday-like external workforce SaaS used by Wolke Systems as a source for worker profile, payroll, benefits, time-off, and lifecycle data. It stays outside the Wolke product boundary and is meant to become a future federation or identity data integration target.
